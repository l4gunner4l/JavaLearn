   <tt>System.out.println("Hello world!");<br/>
   System.out.println("Bye world!");</tt>

2) Что такое "\n"?
- Буква "n"
+ escape-последовательность переводящая каретку на следующую строку
- Знак-энтер

3) Для чего используется Scanner?
- Для сканирования файлов
- Для сканирования лица через камеру
+ Для считывания введённого в консоль текста

4) Что такое %d?
- Константа
+ Спецификатор
- Буква "d"

5) Какой будет вывод?<br/>
<tt>String name = "Tom";<br/>
    int age = 30;<br/>
    float height = 1.7f;<br/>
    System.out.printf("%s  %d  %.2f", name, age, height);</tt>
- %s  %d  %.2f
- name: "Tom"  age: 30  height: 1.7f
+ Tom  30  1.70
- "%s  %d  %.2f", name, age, height
