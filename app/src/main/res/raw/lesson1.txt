Для хранения данных в программе предназначены переменные. Переменная представляет именнованную область памяти, которая хранит значение определенного типа. Каждая переменная имеет тип, имя и значение. Тип определяет, какую информацию может хранить переменная или диапазон допустимых значений.<br/>
<br/>
Переменные объявляются следующим образом:<br/>
<br/>
<b>тип_данных имя_переменной;</b><br/>
<br/>
Например, определим переменную, которая будет называться x и будет иметь тип int:<br/>
<br/>
<tt>int x;</tt><br/>
<br/>
В этом выражении мы объявляем переменную x типа int. То есть x будет хранить некоторое число не больше 4 байт.<br/>
<br/>
В качестве имени переменной может выступать любое произвольное название, которое удовлетворяет следующим требованиям:<br/>
1. Имя может содержать любые алфавитно-цифровые символы, а также знак подчеркивания, при этом первый символ в имени не должен быть цифрой<br/>
2. В имени не должно быть знаков пунктуации и пробелов<br/>
3. Имя не может быть ключевым словом языка Java<br/>
4. Кроме того, при объявлении и последующем использовании надо учитывать, что Java - регистрозависимый язык, поэтому следующие объявления int num; и int NUM; будут представлять две разных переменных.<br/>
<br/>
Объявив переменную, мы можем присвоить ей значение:<br/>
<br/>
<tt>int x;      // объявление переменной<br/>
x = 10;     // присвоения значения<br/>
System.out.println(x);  // 10</tt><br/>
<br/>
Также можно привоить значение переменной при ее объявлении. Этот процесс называется инициализацией:<br/>
<br/>
<tt>int x = 10;     // объявление и инициализация переменной<br/>
System.out.println(x);  // 10</tt><br/>
<br/>
Если мы не присвоим переменной значение до ее использования, то мы можем получить ошибку, например, в следующем случае:<br/>
<br/>
<tt>int x;<br/>
System.out.println(x);</tt><br/>
<br/>
Через запятую можно объявить сразу несколько переменных одного типа:<br/>
<br/>
<tt>int x, y;<br/>
x = 10;<br/>
y = 25;<br/>
System.out.println(x);  // 10<br/>
System.out.println(y);  // 25</tt><br/>
<br/>
Также можно их сразу инициализировать:<br/>
<br/>
<tt>int x = 8, y = 15;<br/>
System.out.println(x);  // 8<br/>
System.out.println(y);  // 15</tt><br/>
<br/>
Отличительной особенностью переменных является то, что мы можем в процессе работы программы изменять их значение:<br/>
<br/>
<tt>int x = 10;<br/>
System.out.println(x);  // 10<br/>
x = 25;<br/>
System.out.println(x);  // 25</tt><br/>
<br/>
<br/>
<b>Ключевое слово var</b><br/>
<br/>
Начиная с Java 10 в язык было добавлено ключевое слово var, которое также позволяет определять переменную:<br/>
<br/>
<tt>var x = 10;<br/>
System.out.println(x);  // 10</tt><br/>
<br/>
Слово var ставится вместо типа данных, а сам тип переменной выводится из того значения, которое ей присваивается. Например, переменой x приваивается число 10, значит, переменная будет представлять тип int.<br/>
<br/>
Но если переменная объявляется с помощью var, то мы обязательно должны инициализировать ее, то есть предоставить ей начальное значение, иначе мы получим ошибку, как, например, в следующм случае:<br/>
<br/>
<tt>var x;      // ! Ошибка, переменная не инициализирована<br/>
x = 10;</tt><br/>

<b>Константы</b><br/>
<br/>
Кроме переменных, в Java для хранения данных можно использовать константы. В отличие от переменных константам можно присвоить значение только один раз. Константа объявляется также, как и переменная, только вначале идет ключевое слово final:<br/>
<br/>
<tt>final int LIMIT = 5;<br/>
System.out.println(LIMIT);  // 5<br/>
// LIMIT=57; // так мы уже не можем написать, так как LIMIT - константа</tt><br/>
<br/>
Как правило, константы имеют имена в верхнем регистре.<br/>
<br/>
Константы позволяют задать такие переменные, которые не должны больше изменяться. Например, если у нас есть переменная для хранения числа pi, то мы можем объявить ее константой, так как ее значение постоянно.<br/>
